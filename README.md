# School-File
此儲存庫為資工二甲的資訊相關課程檔案
## 檔案結構
> 第一階為語言類型 \
> <mark>例:C_C++、web</mark>

> 第二階為專案名稱(若是課程進度應使用日期或單元名稱命名) \
> <mark>例:test、單元1、23.09.30、4</mark>

> 第三階為專案原始檔或專案細分資料夾 \
> <mark>例:main.cpp、main.exe</mark> \
> <mark>例:第一小題、第二小題、(1)、(2)</mark>
```
root
├─C_C++
│  ├─4
│  │  ├─(1)
│  │  ├─(2)
│  │  ├─(3)
│  │  └─(4)
│  └─test
└─web
    └─phptest
```
